# assets
Public static asset repository for Virtumarc branding and lightweight web resources. Used for hosting logos, icons, and other non-sensitive assets for email templates, documentation, and client-facing materials.

Purpose

This repository serves as the centralized source for Virtumarc public-facing static assets. It is intended for lightweight, highly available content that needs to be referenced via URL.

Intended Use

This repo is used to host:
	•	Company logos (email templates, signatures, ticket responses)
	•	Icons and small UI assets
	•	Public images for documentation or portals
	•	Lightweight PDFs or one-pagers

All assets in this repository are public and non-sensitive.

⸻

Delivery Method

Assets are served via:
	•	GitHub Pages
	•	Cloudflare (if fronted via assets.virtumarc.com)
  
⸻

Guidelines
	•	Keep files optimized and small (target <100KB when possible)
	•	Use PNG (transparent) or SVG for logos/icons
	•	Maintain consistent naming conventions
	•	Do not store sensitive or internal-only files
	•	Do not use this repository for scripts, installers, or RMM dependencies

⸻

Versioning
	•	Files may be overwritten to maintain stable URLs (e.g., logo.png)
	•	Breaking changes should use versioned filenames if needed (e.g., logo-v2.png)

⸻

Ownership
